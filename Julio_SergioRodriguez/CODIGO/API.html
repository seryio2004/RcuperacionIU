API Documentation

Core Classes

  • DOM_class - creates tables and forms. Methods: createForm(tipo), load_data(values).
  • EntidadAbstracta - base entity. Methods: inicializar(), ADD(), EDIT(), DELETE(), SEARCH(), change_value_IU(valor).
  • Dom_validations - validates forms. Methods: load_validations(accion), submit_test(accion).
  • ExternalAccess - communication with back. Method: back_request(form, ctrl, action, extra).
  • test - executes entity tests. Method: test_run().
  • validacionesatomicas - atomic validation helpers.

Classes grouped by purpose:

  • Tests: test, Dom_validations
  • HTML generation: DOM_class, EntidadAbstracta
  • Validations: validacionesatomicas
